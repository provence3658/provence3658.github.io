不要62

解题思路：
输入n和m，求[n,m]内的幸运数个数，分解成求[0,n-1]和[0,m]内的幸运数，然后相减。
1、先求从0到i位数的幸运数，如i=3即求[0,999]的幸运数

用dp[i][0] 来存储 0 到 i 位数字中不含4和62的数字个数，即幸运数
用dp[i][1] 来存储 0 到 i 位数字中以 2 开头的幸运数。
用dp[i][2] 来存储 0 到 i 位数字中的非幸运数，即包含4或者62的数字。

龟兔赛跑

解题思路：
因为兔子所需的时间很容易就得出，所以本题的关键是得出乌龟到达终点所需的最短时间。

  • 把起点和终点以及路途中的充电桩的位置记录在数组p中，p[0]为起点，p[n+1]为终点
  • 用数组dp[i]保存乌龟从起点到p[i]这个点所需的最短时间
  • dp[i] = min( (dp[0] + 从起点充满电到p[i]所需的时间),
    ​ (dp[1] + 从第一个充电桩充满电到p[i]所需的时间),
    ​ …
    ​ (dp[i-1] + 从第i-1个充电桩充满电到p[i]所需的时间) )
  • 最后得到的dp[n+1]就是乌龟到达终点所需的最短时间
