二叉查找树的定义

二叉查找树的递归定义如下：

  1. 要么二叉查找树是一棵空树。
  2. 要么二叉查找树由根结点、左子树、右子树组成，其中左子树和右子树都是二叉查找树，且左子树上所有结点的数据域均小于或等于根结点的数据域，右子树上所有结点的数据域均大于根结点的数据域。

二叉查找树的基本操作

  1. 查找操作
    • 如果当前根结点root为空，说明查找失败。
    • 如果需要查找的值x等于当前根结点的数据域root->data，说明查找成功。
    • 如果需要查找的值x小于当前根结点的数据域root->data，说明应该往左子树查找。
    • 如果需要查找的值x大于当前根结点的数据域root->data，说明应该往右子树查找。
  1. 插入操作
    插入操作时，应该寻找插入的值应该在所在的位置，如果在查找的时候发现已经有相同值的结点，说明结点已经存在，如果查找失败，则在查找失败的地方插入该结点。
  2. 二叉查找树的建立
    建立一棵二叉树就是插入n个结点的过程。
  3. 二叉查找树的删除
    删除操作的思路如下：
  4. 如果当前结点root为空，说明不存在权值为给定值的结点，直接返回。
  5. 如果当前结点root的权值恰为给定的权值x，说明找到了想要删除的结点，此时进行删除：
    • 如果当前结点root不存在孩子结点，说明是叶子结点，直接删除。
    • 如果当前结点root存在左孩子，那么在左子树中寻找权值最大的结点pre，然后让pre的数据覆盖root，接着在左子树中删除结点pre。
    • 如果当前结点root存在右孩子，那么在右子树中寻找权值最小的结点next，然后让next的数据覆盖root，接着在右子树中删除结点next。
  6. 如果当前结点root的权值大于给定的权值x，则在左子树中递归删除权值为x的结点。
  7. 如果当前结点root的权值小于给定的权值x，则在右子树中递归删除权值为x的结点。

1043 Is It a Binary Search Tree

【题目链接】
1043 Is It a Binary Search Tree
【题意】
给出N个正整数来作为一棵二叉排序树的插入序列，问着串序列是该二叉树或该二叉树的镜像树的先序序列吗？如果是，则输出YES，并输出对应二叉树或镜像树的后序序列；如果不是，输出NO。
【解题思路】

  • 根据给定序列建立一棵二叉排序树
  • 将该二叉树的先序序列记录到pre数组中，将pre数组与输入的arr数组比较。
  • 将该镜像二叉树的先序序列记录到mirror数组中，将mirror数组与输入的arr数组比较。（镜像二叉树的先序序列只需要先遍历右子树，再遍历左子树）
  • 如果pre数组与arr数组相同，则输出二叉树的后序序列。
  • 如果pre数组与arr数组不同，mirror数组与arr数组相同，则输出镜像二叉树的后序序列。镜像二叉树的后序序列只需要先遍历右子树，再遍历左子树）
  • 如果pre数组和mirror都与arr数组不同，则输出NO。

1064 Complete Binary Search Tree

【题目链接】
1064 Complete Binary Search Tree
【题意】
给出N个非负整数，要用它们构建一棵完全二叉排序树，输出这棵二叉排序树的层序遍历序列。
【解题思路】

  • 可以使用数组来存放完全二叉树，那么对于完全二叉树中的任何一个结点（设编号为x，根结点编号为1），其左孩子结点的编号为2x，右孩子结点的编号为2x+1。那么就可以开一个数组按层序来存放完全二叉树的n个结点。
  • 对于一棵二叉排序树来说，其中序遍历序列是递增的，那么就可以先把输入的数字按从小到大排序，然后对二叉树的结点编号进行中序遍历，并在遍历过程中将数字从小到大填入数组，最后就能得到一棵完全排序二叉树。

【题目链接】
1099 Build A Binary Search Tree
【题意】
二叉树有N个结点，给出每个结点的左右孩子编号（不存在用-1表示）。接着给出一个N个整数的序列，需要把这N个整数填入二叉树的结点中，使之成为一棵二叉排序树。输出这棵二叉排序树的层序遍历序列。
【解题思路】
由于本题直接给了结点编号的关系，所以使用二叉树的静态写法会比较方便。
对于一棵二叉查找树来说，中序遍历是递增的，所以只需要先将数组按从小到大排序，再对这课二叉树进行中序遍历，并在中序遍历的过程中把结点的值填入进去，最后再进行层序遍历即可。
