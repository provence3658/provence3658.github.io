【PAT A1020】Tree Traversals

【题目描述】
Suppose that all the keys in a binary tree are distinct positive integers. Given the postorder and inorder traversal sequences, you are supposed to output the level order traversal sequence of the corresponding binary tree.

【输入格式】
Each input file contains one test case. For each case, the first line gives a positive integer N (≤30), the total number of nodes in the binary tree. The second line gives the postorder sequence and the third line gives the inorder sequence. All the numbers in a line are separated by a space.
【输出格式】
For each test case, print in one line the level order traversal sequence of the corresponding binary tree. All the numbers in a line must be separated by exactly one space, and there must be no extra space at the end of the line.

【输出样例】

【题意】
给出一棵二叉树的后序遍历序列和中序遍历序列，求这课二叉树的层序遍历序列。

【解题思路】

  1. 用后序遍历序列和中序遍历序列来建二叉树，后序序列的最后一个元素是根结点，然后在中序序列中找到这个结点的位置，得知，左子树和右子树分别有几个结点，再递归分别建左子树和右子树即可。
  2. 层序遍历很简单，用队列就可实现。

【PAT A1086】Tree Traversals Again

题目链接：Tree Traversals Again
【题意】
用栈来模拟一棵二叉树的先序和中序遍历过程，求这课二叉树的后序遍历序列
【样例解释】
Push的次序为1、2、3、4、5、6，因此先序遍历为123456
Pop的次序为3、2、4、1、6、5，因此中序遍历为324165
则后序遍历为342651
【解题思路】

  1. 用先序遍历序列和中序遍历序列来建二叉树，先序序列的第一个元素是根结点，然后在中序序列中找到这个结点的位置，得知，左子树和右子树分别有几个结点，再递归分别建左子树和右子树即可。
  2. 后序遍历很简单，用递归就可实现。

【PAT A1102】Invert a Binary Tree

【题目链接】Invert a Binary Tree
【题意】
二叉树有N个结点，给出每个结点的左右孩子结点的编号，把该二叉树反转，输出反转后的二叉树的层序遍历和中序遍历序列。
【解题思路】
由于题目给出的是结点编号的关系，因此使用二叉树的静态写法会非常方便。
首先处理输入问题，如果是数字，则直接把left或right记为该数字，如果是’-‘，则记为-1.同时还要找二叉树的根结点，没有输入过的数字就是根结点。
反转二叉树可以使用后序遍历，在后序遍历访问根结点时交换left和right即可。
